#ESCA Byte Portfolio

A personal portfolio built with Next.js and TypeScript showcasing projects, skills, and a YouTube feed. Clean, component-driven layout using the Next.js App Router.

## Demo

- Local: http://localhost:3000

## Key Features

- Responsive, accessible portfolio layout
- Sections: `Hero`, `Navbar`, `About`, `Projects`, `TechArsenal`, `YouTubeFeed`, `Contact`
- Built with the Next.js App Router and TypeScript
- Modular components in `src/components` for easy customization

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- PostCSS / Tailwind (project includes PostCSS config)

## Project Structure (important files)

- `app/` — Next.js app routes and global layout
- `src/components/` — UI components (About, Projects, Hero, etc.)
- `src/hooks/` — custom hooks
- `src/lib/` — utilities

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm start
```

Open http://localhost:3000 to view locally.

## Deployment

Deploy to Vercel for best compatibility with Next.js, or use any platform that supports Node.js and Next.js static/SSR builds.
